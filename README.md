# Shift Four Properties Client

Static landing page for seller lead capture.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure backend API URL in `scripts/config.js`.

3. Start local static server:
   ```bash
   npm start
   ```

Default local URL: `http://127.0.0.1:8080`

## API Configuration

In `scripts/config.js`:

- Use empty string (`''`) for same-origin deployment.
- Use full backend URL (for example `https://api.your-domain.com`) when frontend and backend are hosted separately.
